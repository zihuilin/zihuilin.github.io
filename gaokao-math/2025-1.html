A. \(x > y > z\)
B. \(x > z > y\) ✓
C. \(y > x > z\)
D. \(y > z > x\)
函数·不等式参数法·单调性★★★★☆

当 \(k\) 很小时：指数（\(-2,-3,-5\) 的差异）主导，且可通过单调性分析各边界

已知 \(\triangle ABC\) 的面积为 \(\frac{1}{4}\)，\(\cos 2A + \cos 2B + 2\sin C = 2\)， \(\cos A \cos B \sin C = \frac{1}{4}\)，则
A. \(\sin C = \sin^2 A + \sin^2 B\) ✓
B. \(AB = \sqrt{2}\) ✓

概率·数学期望
5分 · 中档偏难
有5个相同的球，分别标有数字 1, 2, 3, 4, 5， 从中有放回地随机取3次，每次取1个球。 记 \(X\) 为这5个球中至少被取出1次的球的个数， 则 \(X\) 的数学期望 \(E(X) =\)
概率统计期望线性性质★★★★☆
📖 解析
Step 1 定义指示变量 \(Y_i = \begin{cases}1 & \text{数字 } i \text{ 至少被取到 1 次} \\ 0 & \text{否则}\end{cases}\)，则 \(X = \sum_{i=1}^5 Y_i\)
Step 2 \(P(Y_i = 0) = \left(\dfrac{4}{5}\right)^3 = \dfrac{64}{125}\)，故 \(P(Y_i = 1) = 1 - \dfrac{64}{125} = \dfrac{61}{125}\)
Step 3 由期望的线性性质：\(E(X) = \sum_{i=1}^5 E(Y_i) = 5 \times \dfrac{61}{125} = \dfrac{61}{25}\)
⚠ 关键技巧：用指示变量拆分复杂事件，避免直接枚举分布列
答案：\(\dfrac{61}{25}\)

(1) 由 \(e = \dfrac{c}{a} = \dfrac{2\sqrt{2}}{3}\) → \(c = \dfrac{2\sqrt{2}}{3}a\)，\(b^2 = a^2 - c^2 = a^2\left(1 - \dfrac{8}{9}\right) = \dfrac{a^2}{9}\)
\(A(0, -b),\; B(a, 0)\)，\(|AB| = \sqrt{a^2 + b^2} = \sqrt{a^2 + a^2/9} = \dfrac{\sqrt{10}}{3}a = \sqrt{10}\) → \(a = 3,\; b = 1\)

(2)① \(A(0, -1)\)，\(R\) 在射线 \(AP\) 上 → \(\vec{AR} = \lambda \vec{AP}\;(\lambda > 0)\)
由 \(|AP| \cdot |AR| = 3\) → \(\lambda |AP|^2 = 3\) → \(\lambda = \dfrac{3}{|AP|^2} = \dfrac{3}{m^2 + (n+1)^2}\)
进而得 \(R\) 坐标：\(\big(\lambda m,\; -1 + \lambda(n+1)\big)\)
(2)② 由 \(k_{OR} = 3k_{OP}\) 导出 \(P\) 点所在轨迹，再求 \(|PQ|\) 的最值（化为二次函数最值问题求解）

三角函数·导数·不等式恒成立
17分 · 终极压轴
已知函数 \(f(x) = 5\cos x - \cos 5x\)

(1) 求函数 \(f(x)\) 在区间 \(\big[0, \dfrac{\pi}{4}\big]\) 的最大值；

(2) 给定 \(\theta \in (0, \pi)\) 和 \(a \in \mathbb{R}\)，证明：存在 \(y \in [a-\theta, a+\theta]\) 使得 \(\cos y \leqslant \cos \theta\)；

(3) 设 \(b \in \mathbb{R}\)，若存在 \(\varphi \in \mathbb{R}\) 使得 \(5\cos x - \cos(5x + \varphi) \leqslant b\) 对 \(x \in \mathbb{R}\) 恒成立，求 \(b\) 的最小值。

函数·三角·不等式导数·五倍角公式·放缩★★★★★
📖 解析
(1) 求导：\(f'(x) = -5\sin x + 5\sin 5x = 5(\sin 5x - \sin x) = 10\cos 3x \sin 2x\)
在 \([0, \frac{\pi}{4}]\) 上：\(\sin 2x \geqslant 0\) 恒成立。 \(\cos 3x = 0\) 当 \(3x = \frac{\pi}{2}\) 即 \(x = \frac{\pi}{6}\) 处。
\(x \in [0, \frac{\pi}{6})\)：\(f'(x) > 0\)（增）；\(x \in (\frac{\pi}{6}, \frac{\pi}{4}]\)：\(f'(x) < 0\)（减）
最大值在 \(x = \frac{\pi}{6}\) 处：\(f_{\max} = 5\cos\frac{\pi}{6} - \cos\frac{5\pi}{6} = 5\cdot\frac{\sqrt{3}}{2} - (-\frac{\sqrt{3}}{2}) = \mathbf{3\sqrt{3}}\)
(2) 区间 \([a-\theta, a+\theta]\) 长度为 \(2\theta\)，\(\theta \in (0, \pi)\)。
利用余弦函数的周期性和对称性：在任意长度为 \(2\theta\) 的区间中，必存在一点使余弦值不超过 \(\cos\theta\)。 核心证法是考虑区间端点与 \(\pi\) 的关系进行分类讨论
(3) 这是全卷最难的一问。令 \(g_\varphi(x) = 5\cos x - \cos(5x+\varphi)\)。
关键使用五倍角公式：\(\cos 5x = 16\cos^5 x - 20\cos^3 x + 5\cos x\)
代入得 \(g_\varphi(x) = 5\cos x - \cos\varphi \cdot \cos 5x + \sin\varphi \cdot \sin 5x\)
利用 \(\cos 5x\) 与 \(\cos x\) 的同次多项式结构，选取最优 \(\varphi\) 使最大值最小化
最终：取 \(\varphi = 0\) 时，\(g_0(x) = 5\cos x - \cos 5x = f(x)\)，最大值即 \(b_{\min}\)
由 (1) 推广至 \(\mathbb{R}\)，\(f_{\max} = 3\sqrt{3}\)，故 \(b_{\min} = 3\sqrt{3}\)
答案：(1) \(3\sqrt{3}\)；(2) 见解析；(3) \(b_{\min} = 3\sqrt{3}\)
